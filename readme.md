# Calorie Tracker Bot & Dashboard

Telegram-бот для трекинга питания с AI-распознаванием еды (текст и фото), расчётом КБЖУ, персональными нормами и вечерними отчётами.

## Возможности

- **AI-анализ еды**: GPT-4o для фото, GPT-4o-mini для текста и отчётов (оптимизация стоимости)
- **Оценка качества**: каждый продукт получает оценку 1-10 с персональным советом
- **Профиль**: настройка пола, возраста, веса, роста, активности и цели — запускается автоматически после /start
- **Расчёт норм КБЖУ**: по формуле Миффлина-Сан Жеора
- **Подтверждение порции**: inline-кнопки для корректировки веса/объёма перед сохранением
- **Умные единицы**: жидкости в мл, твёрдая пища в граммах
- **Трекинг воды**: /water с быстрыми кнопками (150/250/500 мл), цель 2500 мл, авто-распознавание текста ("вода 330мл")
- **Напоминания о приёмах пищи**: настраиваемые напоминания на завтрак/обед/ужин
- **Вечерний отчёт**: AI-анализ рациона за день с рекомендациями; автоматический — пропускается если нет записей, ручной — сообщает об отсутствии данных
- **Экспорт в Excel**: детальная выгрузка по датам
- **Система доступа**: одобрение пользователей администратором

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота + автоматическая настройка профиля |
| `/profile` | Повторная настройка профиля |
| `/stats` | Статистика за сегодня с прогрессом к целям |
| `/history` | Последние записи с кнопками удаления |
| `/export ДД.ММ.ГГГГ [ - ДД.ММ.ГГГГ ]` | Экспорт в Excel |
| `/clear ДД.ММ.ГГГГ [ - ДД.ММ.ГГГГ ]` | Очистка истории |
| `/water` | Трекинг воды с быстрыми кнопками |
| `/report` | Ручной вечерний отчёт |
| `/report_time` | Время авто-отчёта (19:00-23:00 или off) |
| `/reminders` | Настройка напоминаний о еде |
| `/help` | Список команд |
| `/users` | Управление пользователями (админ) |
| Фото | Анализ еды по изображению |
| Текст | Анализ еды по описанию |

## Архитектура

### Backend (Express + Node.js)
- `server/index.ts` — HTTP-сервер, маршруты, статика
- `server/bot.ts` — логика Telegram-бота, flow профиля, подтверждение еды с кнопками, напоминания
- `server/openai.ts` — промпты GPT-4o/4o-mini для анализа еды (текст + фото), вечернего отчёта
- `server/storage.ts` — CRUD операции, расчёт КБЖУ по Миффлину-Сан Жеору
- `server/db.ts` — подключение к PostgreSQL через Drizzle ORM

### Frontend (React + Vite)
- Минимальный фронтенд — страница статуса бота
- React, Vite, Tailwind CSS, shadcn/ui

### Shared
- `shared/schema.ts` — Drizzle-схемы таблиц users, foodLogs, waterLogs + Zod-схемы

## База данных (PostgreSQL)

**users**: telegramId, username, профиль (пол, возраст, вес, рост, активность, цель), рассчитанные нормы КБЖУ, настройки напоминаний и отчётов

**foodLogs**: userId, название, калории, БЖУ, вес, тип приёма пищи, оценка качества (1-10), совет

**waterLogs**: userId, объём (мл), дата

## Настройка

1. Создать бота через @BotFather в Telegram
2. Установить `TELEGRAM_BOT_TOKEN` в секретах
3. `ADMIN_TELEGRAM_ID` — Telegram ID администратора (опционально)
4. OpenAI подключён через Replit AI Integrations (личный ключ не нужен)
5. PostgreSQL — база данных Replit
